# UI 还原开发模板

用于根据设计稿、截图或样式描述还原页面/组件 UI 及基础交互。

## 增强提示词结构

【做什么】：根据[设计稿链接/截图描述]还原[页面/组件名称，如：个人中心页面、商品卡片组件]的UI及基础交互。

【怎么做】：

1. 样式实现：使用[CSS3/TailwindCSS/SCSS]编写样式，严格匹配设计稿的[尺寸/颜色/字体，如：主色#165DFF、字体大小14px、边距8px]；
2. 布局方式：采用[Flex/Grid/定位]布局，保证元素位置与设计稿偏差≤1px；
3. 交互实现：[交互逻辑，如：hover状态切换背景色、点击卡片跳转至详情页]；
4. 适配处理：针对[移动端/PC端]做响应式适配，设计稿未标注的适配规则遵循[750px设计稿1rem=100px/PC端最小宽度1200px]。

【做到何种程度】：

1. 视觉还原：UI视觉效果与设计稿一致，无明显偏差，[关键元素，如：按钮圆角、文字行高]100%匹配；
2. 交互效果：基础交互流畅，无延迟/卡顿，[特殊效果，如：渐变色、阴影]符合设计要求；
3. 兼容性：在[指定设备/浏览器]上样式无错乱，适配效果符合预期；
4. 代码交付：样式代码解耦，无冗余，可复用性强。
